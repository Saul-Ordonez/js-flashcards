JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation. Such as addition subtraction ect....

Variables

Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

Variable Naming Conventions

The proper use of words when naming variables.

Functions

A subprogram designed to perform a particular task.

Methods

Actions that can be performed on objects.

Parameters

Variables listed as part of the function definition.

Arguments

The actual value of the parameter that gets passed to function.

Return

Store result of the function.

Chaining Methods

Common syntax for invoking multiple method calls.

Strings

A location to store a series of characters.

Boleans

A datatype that returns either of two values i.e. true or false.

Undefined

No value has been explicitly assigned to the variable.

NaN

Meaning 'Not a Number', this indicates that the value is not a legal number.

Escape

Encodes a string. This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

Alert

A method that displays a pop-up box in your webpage.
